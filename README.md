# Becoming of BKMNG

Ongoing documentation of the BKMNG project — decisions, directions, journal entries, and the evolution of thinking as the project develops.

This is a living record: not historical archive, but active working documentation.

## Structure

- `journal/` — Dated entries on progress, ideas, and shifts in thinking
- `decisions/` — Key project decisions and their rationale
- `directions/` — Exploration of potential directions and open questions
